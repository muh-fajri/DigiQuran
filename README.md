# DigiQuran

**DigiQuran** is an online Quran application which is can be read without any boundary of time and places as long an internet connection is available.

## Feature

* List surah
* List content (arabic, transliteration, translation)

Currently added:
* Searching
* Available in 2 languages (Indonesian and English)

## Feature Changing
* Change float to flexbox layout
* Spinner loading to inform user that page is loading
* Limiting initial ayah to speed up of loading page
* Highlight searching result to inform user which surah or ayah containing the keyword

## Built with
* HTML and CSS
* JavaScript (using jQuery Library)
* Quran REST Server API
